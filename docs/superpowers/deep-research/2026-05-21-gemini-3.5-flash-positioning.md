## TL;DR (researcher's own summary)

- `gemini-3.5-flash` GA May 19 2026. $1.50 / $9 per 1M tokens. 1M-token context. 76.2% Terminal-Bench 2.1, 83.6% MCP Atlas.
- Google's verbatim marketing phrases: *"sub-agent deployment," "long-horizon agentic tasks," "iterative coding cycles," "thought preservation."* These map 1:1 to the Cerebral Valley problem statement. Pitch must repeat them.
- To claim 3.5 Flash as centerpiece: **demote Managed Agents/Antigravity to a single optional Containment step, refactor Forensics into 3–5 parallel 3.5 Flash sub-agents** (Frame Forensics, Voice-Print, Reverse Provenance, Counter-Strategy, Regulatory Precedent) with visible streaming chains-of-thought.

Mitigations: pin `thinkingLevel: "low"` on workers, cap ≤4 parallel calls, cache hot-swap fallback at 12s budget.

## Live-demo failure modes (GitHub-verified)

| Failure | Source | Mitigation |
|---|---|---|
| ADK parallel function-call count mismatch | github.com/ag-ui-protocol/ag-ui/issues/1334 | Don't use ADK Python LRO; orchestrate fan-out in Next.js with `Promise.all` + direct `@google/genai`. |
| Sub-agent timeout silent fallback to lower-tier Flash | github.com/google-gemini/gemini-cli/issues/24412 | Hard-pin `model: "gemini-3.5-flash"`; never `gemini-flash-latest`; log model name returned. |
| Gemini 3 Pro 600s idle timeout (Flash inherits) | github.com/openclaw/openclaw/issues/76071 | `AbortController` at 12s per worker; surface "fallback to cached trace" not hang. |
| API hangs 5 min on Node undici default | github.com/google-gemini/gemini-cli/issues/18030 | `signal: AbortSignal.timeout(15000)`. |
| Search-grounded latency spikes | I/O migration doc · datacamp.com | Cap grounded to 2/5 sub-agents (Reverse Provenance + Regulatory). |
| **Silent `thinking_level: high → medium` regression on migration** | ai.google.dev/gemini-api/docs/whats-new-gemini-3.5 | Explicit `thinkingLevel: "low"` on workers, `"medium"` orchestrator, `"high"` verdict. |
| Cost inflation: high-thinking 3.5 Flash ~5.5× 3 Flash Preview reasoning | nxcode.io | `thinkingLevel: "low"` everywhere except verdict; `cachedContent` ($0.15/M) for system prompts. |

## Pitch-language guardrails

**The "Image Analyzer" trap.** Cerebral Valley reuses a banned-list across events (verified on the LandingAI Financial AI Hackathon Championship Luma page `luma.com/jme15h1t`): "AI mental health advisor; Projects giving medical advice; Basic RAG applications; Simple Streamlit only applications; Image analyzers; 'AI for education' chatbots; AI job application screeners; AI nutrition coach; Personality analyzers." The Google I/O Hackathon page doesn't publish its own banned list publicly, but assume the LandingAI list applies.

Reframe required:
- Lead with **"enterprise wire-fraud kill-switch"** — never "deepfake detector"
- Show wire-freeze action in first 30 seconds; action > analysis
- "Frame Forensics" not "image classifier"
- Avoid: "classifier," "detector," "analyzer"
- Use: "kill-switch," "forensics agent," "investigator," "containment layer"

## Recommendations (researcher's own)

1. Re-read `ai.google.dev/gemini-api/docs/whats-new-gemini-3.5` end-to-end; highlight the verbatim phrases; paste into README and pitch.
2. Pin `model: "gemini-3.5-flash"` everywhere; explicit `thinkingLevel` on every call; log returned model name (catch silent downgrades).
3. By end of Thursday: 5 sub-agents return distinct JSON; agent-graph lights up; orchestrator gates verdict on 3-of-5 (not 5-of-5) so any single sub-agent crash doesn't kill the demo.
4. Friday: Search grounding + containment + $50M end-card + **3 full end-to-end dry runs on real café Wi-Fi** (not home fiber).
5. Saturday 10:30 AM on the venue Wi-Fi: one more dry run before judging.
6. Never say "image analyzer," "image classifier," "deepfake detector" in first 30 seconds. Use "wire-fraud kill-switch," "synthetic-media containment layer," "enterprise SaaS."
7. Keep one **always-live, never-cached** sub-agent (recommend Counter-Strategy — cheapest, lowest-failure) for live-demo points.
8. Submit repo URL + 60s screen-capture as backup. Vercel deploy on same domain so judges can poke during deliberation.

## Open caveats

- Knowledge-cutoff conflict (Jan 2025 vs Jan 2026) — primary doc wins.
- Armadin token-efficiency claim conflicts (72% vs 68%) — say "over 40% better … at a fraction of tokens."
- No verifiable winner-pitch examples explicitly naming 3.5 Flash yet — model has been GA for ~48 hours. Green-field; set the pattern.
- 93-sub-agent Antigravity demo was Google's internal telemetry, not independently reproduced.
- 3.5 Flash does NOT support Computer Use or image segmentation. Don't promise either.
- Live Demo 45% weight is unusually heavy — optimize for robust theater. Every visible element needs hot-swap cached fallback. Every API call needs 12-second AbortController. Agent graph must render even if every backend call fails.
